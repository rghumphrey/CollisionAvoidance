Flashing Raspberry PI OS (64-bit) Raspberry Pi using Raspberry Pi Imager

  1. Get the Raspberry Pi Imager from here
  2. Insert your microSD card into your computer with a microSD adapter.
  3. Use the Imager to select the Raspberry PI OS (64-bit) and select the SD card as storage
  4. Ensure that SSH is enabled in the advanced settings, and set username and password.
  5. After flashing, you can insert the SD card into the Pi. From here you will have to use a microHDMI-to-HDMI cord to connect to a screen to connect the Pi to a network.
  6. From here, follow the set up in networking.txt

Installing dronekit and mavlink with Patches

Follow these steps to install dronekit and mavlink, and apply the necessary patches:

sudo python3 -m pip uninstall dronekit pymavlink -y
mkdir -p updated_dronekit_pkgs && cd updated_dronekit_pkgs
git clone https://github.com/morzack/dronekit-python.git && cd dronekit-python
git checkout dronekit-ignore-non-vehicle-heartbeats
sudo python3 -m pip install . -v
cd ..
sudo python3 -m pip uninstall pymavlink -y
git clone https://github.com/ArduPilot/mavlink.git && cd mavlink
git submodule update --recursive --init
cd pymavlink
git pull origin master
git checkout 67669c3
sudo python3 setup.py install
